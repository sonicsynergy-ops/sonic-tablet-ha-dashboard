# Hemma reference material

> **Reference only. Do not install these files into Home Assistant.**

This directory contains the preserved Home Assistant/Lovelace material from [Hemma](https://github.com/willsanderson/Hemma) by willsanderson.

It is retained for visual, interaction, theme, and template study while Sonic Synergy Studio develops its own APK-backed tablet dashboard. Files here are not the Sonic application runtime and must not be treated as installation instructions or copied into a live Home Assistant configuration.

The preserved reference includes:

- The original Hemma project README in [`UPSTREAM_README.md`](UPSTREAM_README.md)
- The example Hemma Lovelace dashboard and button-card templates
- Home Assistant helper, sensor, theme, and resource configuration
- Hemma fonts, icons, backgrounds, weather assets, and scripts

Relevant design ideas may be reimplemented in Sonic-owned application components. Home-specific assumptions and Lovelace/HACS dependencies are not part of the target architecture.
